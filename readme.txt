== Description ==

Detects mailto: links and converts them into contact forms that send emails via the SquareSend.com API


== Installation ==

1. Upload the `contact-blaster` folder to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress


== Frequently Asked Questions ==

= Do I need a SuqareSend account? =

Yes, you need an account on SquareSend for this plugin to be able to send messages.
